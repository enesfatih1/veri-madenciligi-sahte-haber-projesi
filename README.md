# Sahte Haber Tespiti – Veri Madenciliği Projesi

Bu proje, **Veri Madenciliği** dersi kapsamında hazırlanmıştır.

## Projenin Amacı
Bu çalışmanın amacı, haber metinlerini makine öğrenmesi ve doğal dil işleme
yöntemleri kullanarak **sahte (fake)** veya **gerçek (true)** olarak
sınıflandırmaktır.

## Kullanılan Teknolojiler
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## Proje İçeriği
- Veri ön işleme
- Metin temizleme
- Özellik çıkarımı
- Sınıflandırma modeli
- Model performans değerlendirmesi
